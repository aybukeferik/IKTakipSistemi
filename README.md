# IKTakipSistemi

##Projenin Amacı
###Bu projenin amacı, insan kaynakları süreçlerini dijital ortamda yönetmek için temel bir İK Takip Sistemi geliştirmektir. Sistem; çalışan bilgileri, izin talepleri ve izin onay süreçlerinin kullanıcı ve yönetici rollerine göre yönetilmesini sağlar. Katmanlı mimari yapısı sayesinde veri erişimi, iş mantığı ve kullanıcı arayüzü ayrıştırılarak daha sürdürülebilir ve ölçeklenebilir bir yapı hedeflenmiştir.
